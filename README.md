Developed a full-stack Student Management System web application using Spring Boot, Spring Data JPA, Thymeleaf, HTML, and CSS to efficiently manage student records with a clean UI and secure backend architecture. The system allows administrators to perform complete CRUD operations including adding, updating, and deleting student details while ensuring data persistence using a relational database (MySQL).

Key features include:

 1) Student Registration.
 2) View student list in dashboard.
 3) Update and delete student records.
 4) UI using HTML & CSS
 5) Database integration using Spring Data JPA (Hibernate)
 6) Layered architecture (Controller, Service, Repository)

Depencies used:-
 1) Spring web
 2) Spring Boot DevTool
 3) MySql Driver
 4) Spring Data Jpa
 5) Thymeleaf

Technologies Used:-

. Java (JDK 8 or above) , Java Spring Sts Tool / Eclipse Ide ,  MySql for Database 

How to Run:-

. Clone or download this repository. , Open the project in spring sts tool / eclipse ide . , Make sure you have Java SDK installed. , Run the Main_class.
